#  계획서

1. **무슨 웹사이트**

사용자가 메뉴에서 동물 이름을 클릭하면,  
해당 동물의 사진과 간단한 정보를 볼 수 있는  
 동물 정보 웹사이트를 제작

---

2. **웹사이트 구성**

- **상단 제목**  
  - "동물 정보 백과"라는 사이트 이름 표시

- **좌측 메뉴**  
  - 동물 이름 목록 (예: 고양이, 강아지, 코끼리 등)
  - 클릭하면 해당 동물 정보를 보여주는 기능 포함

- **오른쪽 본문**  
  - 선택된 동물의 이미지와 설명이 표시되는 영역
  - 클릭 전에는 기본 안내 메시지를 보여줌
